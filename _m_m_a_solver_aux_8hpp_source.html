25// MMA solver using a dual interior point method
26//
27// Original code by Niels Aage, February 2013
28// Modified to use OpenMP by Jun Wu, April 2017
29// Various modifications by Jérémie Dumas, June 2017
30//
31// The class solves a general non-linear programming problem
32// on standard from, i.e. non-linear objective f, m non-linear
33// inequality constraints g and box constraints on the n
34// design variables xmin, xmax.
35//
36// min_x^n f(x)
37// s.t. g_j(x) < 0, j = 1,m
38// xmin < x_i < xmax, i = 1,n
39//
40// Each call to Update() sets up and solve the following
41// convex subproblem:
42//
43// min_x sum(p0j./(U-x)+q0j./(x-L)) + a0*z + sum(c.*y + 0.5*d.*y.^2)
44//
45// s.t. sum(pij./(U-x)+qij./(x-L)) - ai*z - yi <= bi, i = 1,m
46// Lj < alphaj <= xj <= betaj < Uj, j = 1,n
47// yi >= 0, i = 1,m
48// z >= 0.
49//
50// NOTE: a0 == 1 in this implementation !!!!
51//
53
